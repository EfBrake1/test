Мой первый сайт на Github

Для нового репозитория первое создать репозиторий на сайте Github, следовать инструкциям на сайте "git init"

Для работы с подключёным репозиторием

git add .
Выполните первый коммит:
git commit -m "Init"
git push-для отправки

Для получения файлов из подключёного репозитория

git pull

Как получить доступ к файлам находящихся в репозитории

1.Скачать зип архив сайта репозитория

2.Git clone url -что мы получаем прямой доступ к репозиторию (синхронизация)
